# Portafolio Web

Este repositorio contiene el desarrollo de un proyecto de **portafolio web**.  

⚠️ **Nota importante:**  
El contenido mostrado en esta página, incluyendo datos personales y experiencia laboral, **no representa el currículum real del autor** y puede contener errores o información ficticia. Este proyecto se ha creado únicamente con fines educativos y de práctica en desarrollo web (HTML y CSS).

---

## Estructura del proyecto

El proyecto se compone de los siguientes archivos y carpetas:

- **index.html**: Página principal con la presentación del portafolio, incluyendo una sección de perfil personal y proyectos.
- **cv.html**: Página de currículum con secciones de datos personales, idiomas, formación y experiencia laboral. Toda la información puede ser ficticia y solo sirve para fines de práctica.
- **contacte.html**: Página de contacto con un formulario de contacto funcional en diseño (sin funcionalidad real de envío de datos).
- **style.css**: Archivo de estilos CSS que da formato a todas las páginas del proyecto, incluyendo colores, tipografía, disposición de secciones y responsive design.
- **assets/**: Carpeta que contiene imágenes utilizadas en las páginas (perfil, iconos, fondo, etc.).

---

## Tecnologías utilizadas

- **HTML5**: Estructura semántica de las páginas.
- **CSS3**: Estilos y diseño responsivo.
- **Responsive Design**: Uso de media queries para adaptar el diseño a distintos tamaños de pantalla.
- **Buenas prácticas de accesibilidad y semántica** (uso de etiquetas `<header>`, `<main>`, `<footer>`, `<section>`, etc.).
